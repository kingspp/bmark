# bmark
python benchmarking utility

#### Requirements  
1. **OS:** macOS Maverics+ or Ubuntu 14.04+  
2. **Python:** v3.6.0+ 
3. **Tensorflow:** v1.4.0+ 


#### What is bmark?
Bmark - "Benchmark" is a simple yet intutive process monitoring tool, which can monitor,
1. CPU Usage
2. RAM Usage
3. GPU Usage (Supports only NVidia for now)
